Here you can import users. User import is in a CSV format. Here is an example of valid input:

Name,Email,Team
Stefan Nelson,nels8926@d.umn.edu,Stefan’s Team
Peter Peterson,pahp@d.umn.edu,Peter’s Team
William Effinger,effin015@d.umn.edu,Stefan’s Team
Hugh Cunningham,cunni513@d.umn.edu,Peter's Team

Optionally, you can also include a password column. Otherwise passwords will all be set to password to be changed by users on first login. For example:

Name,Email,Team,Password
Stefan Nelson,nels8926@d.umn.edu,Stefan’s Team,password1!
Peter Peterson,pahp@d.umn.edu,Peter’s Team,Sup3rS3cur3
William Effinger,effin015@d.umn.edu,Stefan’s Team,hunter2
Hugh Cunningham,cunni513@d.umn.edu,Peter's Team,DWiR+EHM0wcU2Mha

These files can easily be generated in LibreOffice Calc (or, presumably, Microsoft Excel or other comparable software) by exporting as CSV with default options. Download a sample to get started.
